Um gráfico de barras só em HTML

Esse é um exemplo utilizando CSS e HTML puro para construir gráficos de barras. Para realizar essa tarefa, utilizamos divs que são estilizadas com a propriedade width para definir quantos píxels cada barra terá. Não precisamos se preocupar com a altura da barra nesse momento, o html já irá inferir um tamanho padrão pelos números que colocamos dentro de cada div.
